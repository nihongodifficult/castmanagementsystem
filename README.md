# 店舗横断キャスト管理アプリ (Windows向け)

複数店舗に在籍する女の子の情報を、1つのデータベースでまとめて管理するためのデスクトップアプリの簡易サンプルです。

## 特徴
- 店舗名・名前・年齢・メモを登録/更新/削除
- 検索 (店舗/名前/メモに部分一致)
- 店舗名はプルダウンで選択
- ローカルのSQLiteデータベースに保存
- Windows標準のPython + Tkinterで動作

## 使い方
1. WindowsにPython 3.10以降をインストールします。
2. このリポジトリを取得して、以下を実行します。

```bash
python app.py
```

初回起動時に `cast_management.db` が作成されます。

## これから追加できる機能例
- 検索/フィルタ (店舗ごと/キーワード)
- 画像登録
- 出勤表・予約状況の管理
- CSVのインポート/エクスポート
